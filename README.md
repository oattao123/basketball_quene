# Basketball Queue App 🏀

แอปพลิเคชัน React Native (Expo) สำหรับจัดการคิวการเล่นบาสเกตบอล

## ฟีเจอร์

- ✅ เพิ่มทีมใหม่เข้าคิว
- ✅ แสดงทีมที่กำลังเล่นอยู่ (2 ทีม)
- ✅ แสดงทีมที่รอคิวถัดไป
- ✅ เริ่มเกมใหม่อัตโนมัติ
- ✅ ลบทีมออกจากคิว
- ✅ จบเกมและเริ่มเกมใหม่

## การติดตั้ง

1. ติดตั้ง dependencies:
```bash
npm install
```

2. รันแอป:
```bash
# เริ่ม Expo Dev Server
npm start

# หรือรันบนแพลตฟอร์มเฉพาะ
npm run android
npm run ios
npm run web
```

## การใช้งาน

1. **เพิ่มทีม**: ใส่ชื่อทีมในช่องข้อความและกดปุ่ม "เพิ่ม"
2. **เริ่มเกม**: เมื่อมีทีมอย่างน้อย 2 ทีม กดปุ่ม "เริ่มเกม"
3. **จบเกม**: เมื่อเกมจบ กดปุ่ม "จบเกม" เพื่อให้ทีมถัดไปเข้าเล่น
4. **ลบทีม**: กดปุ่ม "ลบ" ข้างชื่อทีมเพื่อลบออกจากคิว

## โครงสร้างแอป

- แสดงทีมที่กำลังเล่นอยู่ในรูปแบบ "ทีม A VS ทีม B"
- แสดงจำนวนทีมที่รอคิวทั้งหมด
- แสดงทีมถัดไปที่จะเข้าเล่น
- UI ที่เป็นมิตรกับผู้ใช้และรองรับภาษาไทย
